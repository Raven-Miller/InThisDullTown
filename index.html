In This Dull Town

About the Project

In This Dull Town is an adventure horror fiction project.

It follows a band of 5 teens investigating the paranormal experiences of their hometown, Pine Ridge.

It was first created January 2025 and made available to the public in July 2025.

The project doesn't work in linear time. There is no 1 set sequence of events. There is no defining beginning and end. It's a messy collection of stories from the main characters.

It's not that serious, this is just a silly little project that I'm doing for fun. So, please bear with me.
